# CS2510: Distributed Systems

This repository contains coursework for CS2510: Distributed Systems at the University of Pittsburgh. Each assignment is organized in its own directory, and the repository includes GitHub Actions workflows to automate testing.

## Assignments

- **Assignment 1**: Group RPC Communication System
  - Implemented an asynchronous RPC-based group chat system using Python and gRPC.
  - Supports multiple clients, message history, and unread message tracking.
  - Includes automated tests for delayed client connection and real-time message broadcasting.
