# CS2770: Computer Vision

This repository contains the code for the assignments in CS2770: Computer Vision. The code is organized into different folders for each assignment. Each folder contains a Jupyter notebook with the code for that assignment, as well as any necessary data files.

## Assignments

- **Assignment 1**: Image Classification with PyTorch
  - Implemented a simple image classification model using PyTorch.
  - Explored the SUN20 dataset and visualized the data.
  - Used numerical approximation to compute derivatives for optimization.
